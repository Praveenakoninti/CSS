    9.Create an ordered list of the most common text formatting tags in HTML. Within each list item, use an unordered list to list the specific use cases and best practices for that tag
    1. <strong>:
      • Use case: Indicates strong importance or emphasis.
      • Best practices: Use sparingly and only when the text truly needs to stand out.
    2. <em>:
      • Use case: Indicates emphasis or stress on a particular word or phrase.
      • Best practices: Use sparingly and only when the emphasis is essential to convey the meaning of the content.
    3. <u>:
      • Use case: Underlines the enclosed text.
      • Best practices: Use sparingly and only when there is a specific need to underline the text, such as indicating a link.
    4. <s>:
      • Use case: Renders the enclosed text with a strikethrough line.
      • Best practices: Use to indicate deleted or no longer relevant information, such as in a document revision or history.
    5. <sup>:
      • Use case: Formats the enclosed text as superscript (raised above the baseline).
      • Best practices: Commonly used for footnotes, mathematical equations, and chemical formulas.
    6. <sub>:
      • Use case: Formats the enclosed text as subscript (lowered below the baseline).
      • Best practices: Commonly used for chemical formulas, mathematical equations, and footnotes.
    7. <code>:
      • Use case: Indicates that the enclosed text represents computer code or a programming element.
      • Best practices: Use to format code snippets, programming language keywords, or variable names.
    8. <blockquote>:
      • Use case: Indicates that the enclosed text is a long quotation or a block of text quoted from another source.
      • Best practices: Use to format substantial quotations, ensuring proper citation and indentation.
